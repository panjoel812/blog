## 基本运算符

|        运算符         |               说明               |
| :-------------------: | :------------------------------: |
|     or, and, not      |      布尔或、布尔与、布尔非      |
|      is, is, not      |       判断是否为同一个对象       |
| <, <=, >, >=, ！=, == |     比较值是否相当，可以连用     |
|       \|, ^, &        |     按位成，按位异或、按位与     |
|        <<, >>         |               移位               |
|           ~           |             按位翻转             |
|   +, -, *, /, //, %   | 加，减，乘，浮点除、整数除、取余 |
|          **           |              幂运算              |

#### 1.比较运算符可以连用，并且含义和我们日常使用完全一致

```python
>>> a=4
>>> 3<a<10    #关系运算符可以连用
True
```

#### 2.位操作

```python
>>> a = 0b11001
>>> b = 0b01000
>>> c = a|b
>>> bin(c)       #bin( )可以将数字转成二进制表示
"0b11001"
>>> bin(c&b)
"0b1000"
>>> bin(c^b)
"0b10001"
```

#### 3.加法操作

```python
>>> 3+2
5 
>>> "3"+"2"
"32"
>>> [10,20,30] + [40,50,60]
[10,20,30,40,50,60]
```

#### 4.乘法操作

```python
>>> 3*2
6
>>> "qwq"*3
"qwqqwqqwq"
>>> [10,20,30]*3
[10,20,30,10,20,30,10,20,30]
```
