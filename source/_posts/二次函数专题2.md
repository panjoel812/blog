## 二次函数专题2

### 含参最值讨论问题

#### 课后习题

{% note success %}

已知函数$f(x)=x^2-2ax+4$在区间$[1,4]$上的最小值为-8，求参数$a$的值

 {% endnote %}

接下来我们解这一题

$f(x)=(x-a)^2+4-a^2$

当$a≤1$时，

$f(1)=1-2a+4=5-2a$

$5-2a=-8$

$a=\frac {13}{2}$

$\because a≤1$

$\therefore $舍去

当$1<a<4$时，

$f(a)=4-a^2$

$4-a^2=-8$

$a^2=12$

$a=2 \sqrt 3$

当$a≥4$时，

$f(4)=16-8a+4=20-8a$

$20-8a=-8$

$8a=28$

$a=\frac {7}{2}$

$\because a≥4$

$\therefore $舍去

答：$a=2\sqrt 3$

#### 例题1

{% note success %}

已知函数$f(x)=x^2-2x+4$在区间$[a,a+1]$上，最小值为$7$，求参数$a$的值

 {% endnote %}

$f(x)=(x-1)^2+3$

当$a+1\leq 1\Rightarrow a\leq 0$时

$f(a+1)=a^2+3$

$a^2+3=7$

$a=2$（舍）

当$a<1<a+1$时

$f(1)=3$（舍）

当$a\geq 1$时

$f(a)=a^2-2a+4$

$a^2-2a+4=7$

$a_1=-1$（舍），$a_2=3$

答：$a=3$
